# Das Spiel spielen

Die Seite **Spieler** ist Ihre Zentrale für alle Spiele. Hier können Sie gespeicherte Spiele spielen und Ihren Verlauf einsehen.

## Registerkarten

- **Verfügbare Spiele:** Diese Registerkarte listet alle Spiele auf, die Sie erstellt oder über einen Link geladen haben. Für jedes Spiel haben Sie mehrere Optionen:
    - **Spielen:** Klicken Sie auf das Wiedergabesymbol, um das Spiel sofort zu starten.
    - **PDF herunterladen:** Klicken Sie auf das Download-Symbol, um ein druckbares PDF-Arbeitsblatt zu erstellen. Jedes Level erhält zwei Seiten: eine mit dem Rätsel und den Hinweisen und eine zweite mit dem Lösungsschlüssel. Dies ist perfekt für das Offline-Spielen.
    - **Teilen:** Klicken Sie auf das Teilen-Symbol, um einen eindeutigen Link in Ihre Zwischenablage zu kopieren. Jeder mit dem Link kann das Spiel seiner Bibliothek hinzufügen.
    - **Löschen:** Klicken Sie auf das Papierkorb-Symbol, um das Spiel dauerhaft zu entfernen.

- **Spielverlauf:** Diese Registerkarte zeigt die Ergebnisse jedes abgeschlossenen Spiels, einschließlich ob Sie gewonnen oder verloren haben, wie viele Level Sie abgeschlossen haben und das Datum, an dem Sie gespielt haben.

## Spielablauf

Wenn Sie ein Spiel starten, wechselt die Ansicht zum Spielfeld. Ihr Ziel ist es, alle versteckten Wörter zu finden, bevor die Zeit abläuft. Klicken und ziehen Sie mit der Maus, um Wörter auszuwählen. Wenn die Zeit abläuft, endet das Spiel und alle richtigen Antworten werden auf dem Gitter angezeigt.
